Bem vindo ao Developers Paraná.

A ideia deste movimento é dar o empurrão que faltava para muito dos desenvolvedores paranaenses começar a copartilhar seu conhecimento, através de MeetUps, Coding Dojo, Conferências e tudo mais que pudermos fazer.

Nossa iniciativa, é para auxiliar nas comunidades e movimentos nas suas ações, não competindo com os mesmos.

Como participar

Estamos tomando decisões em conjunto no Slack criado para o Dev Paraná, para fazer parte acesse devparana.herokuapp.com.
